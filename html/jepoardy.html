      Welcome

      Every tag has to be styled individually. If you want to change the font, you have to update every single tag.

      This second paragraph has to repeat all the same styles.

      ``` **Option B — External CSS (clean, reusable):** Save as `index.html`: ```html External CSS

      Welcome

      Change the font once in styles.css and it updates everywhere.

      This paragraph gets the same styles automatically.

      ``` Save as `styles.css` in the same folder: ```css body { font-family: Georgia, serif; } h1 { color: navy; text-align: center; } p { color: gray; font-size: 18px; } ``` > **Try it:** In Option B, change the font in `styles.css` to `Arial` — notice both paragraphs update at once.
